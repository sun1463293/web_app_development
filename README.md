## 系統簡介

現代人常面臨不知道錢花去哪裡的困擾，導致無法有效儲蓄與理財。為了幫助個人更好地管理財務狀況，我們開發了這個「個人記帳系統」。

本系統針對**各種需要記帳的人（包括學生、上班族、家庭主婦等）**，提供了直覺的記帳體驗與清晰的圖表分析，幫助使用者輕鬆掌握每日財務狀況，達成個人的理財與儲蓄目標。
